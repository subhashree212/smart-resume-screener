# Smart Resume Screener
NLP-powered resume screening system built with Python, Flask, and scikit-learn.

## What it does
- Matches resumes against job descriptions using TF-IDF vectorization and cosine similarity
- Achieves 88%+ accuracy in identifying qualified candidates
- REST API endpoint for batch resume uploads
- Reduces manual screening time by 85%

## Tech Stack
Python · Flask · scikit-learn · TF-IDF · SQLite · REST API
